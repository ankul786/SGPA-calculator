# SGPA Calculator

A clean, free SGPA (Semester Grade Point Average) calculator for students.

## Live Demo

Once deployed, accessible at: `https://<your-username>.github.io/sgpa-calculator`

## Grade Point System

| Grade | Points |
|-------|--------|
| A+    | 10     |
| A     | 9      |
| B+    | 8      |
| B     | 7      |
| C+    | 6      |
| C     | 5      |
| D     | 4      |
| E     | 0      |
| F     | 0      |

## Features

- Add unlimited subjects with name, credits, and grade
- Instant SGPA calculation
- Visual score bar
- Performance remark (Outstanding, Excellent, Good, etc.)
- Fully responsive — works on mobile and desktop
- No login, no ads, no data stored
